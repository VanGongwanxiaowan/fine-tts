# AudioProcessor API 音频处理器API

TTS.utils.audio.AudioProcessor is the core class for all the audio processing routines. It provides an API for

TTS.utils.audio.AudioProcessor是所有音频处理程序的核心类。它提供了一个应用程序接口（API），用于

Feature extraction. 特征提取。

Sound normalization. 声音归一化。

Reading and writing audio files. 读取和写入音频文件。

Sampling audio signals. 音频信号采样。

Normalizing and denormalizing audio signals.

音频信号的归一化和反归一化。

Griffin-Lim vocoder. 格里芬-林声码器。

The AudioProcessor needs to be initialized with TTS.config.shared_configs.BaseAudioConfig. Any model config also must inherit or initiate BaseAudioConfig.

AudioProcessor需要使用TTS.config.shared_configs.BaseAudioConfig进行初始化。任何模型配置也必须继承或初始化BaseAudioConfig

win_length（int）——每个音频帧都用长度为`win_length`的窗口进行加窗处理，然后填充零以匹配`fft_size`。默认值为1024。

mel_fmin（浮点型）——用于梅尔基滤波器的最小频率水平。男性声音约为50，女性声音约为95。需要针对数据集进行调整。默认值为0。

mel_fmax（浮点型）——用于梅尔基滤波器的最大频率水平。它需要根据数据集进行调整。

spec_gain（int）——将振幅转换为分贝时应用的增益。默认值为20。

signal_norm（布尔值）——启用/禁用信号归一化。默认为True。
